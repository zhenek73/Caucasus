## Технические требования

- Один HTML-файл, всё inline (CSS в <style>, JS в <script>)
- Полностью адаптивный (mobile-first)
- Плавная прокрутка между секциями
- Sticky header с навигацией
- Анимации при скролле (fade-in через IntersectionObserver)
- Цветовая палитра: 
  - Основной тёмный: #1a1a2e
  - Акцент горный закат: #e94560 / #f5a623
  - Зелёный природный: #2d6a4f
  - Текст: #f8f9fa
  - Карточки: полупрозрачный белый на тёмном
- Шрифты: Google Fonts — "Playfair Display" для заголовков, "Inter" для текста
- Иконки: использовать Unicode-символы или SVG inline (не подключать внешние библиотеки)
- Изображения: использовать красивые gradient placeholder'ы с CSS (имитирующие горные пейзажи) — реальные фото хозяин добавит сам
- Секции разделены плавными SVG-волнами между блоками
